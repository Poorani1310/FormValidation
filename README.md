# FormValidation
Form Validation

Created a simple registration form that consists of the following fields: 
Name, 
Email, and 
Password

• Validate that the Name field is not empty. Display an error message Enter a valid name - maximum 1 character allowed. If any input given then the error message disappears.  

• Validate that the Email field contains a valid email address. Display an error message if the format is incorrect, and disapperars if valid email entered.  

• Validate that the Password field has a minimum length of 6 characters. Display an error message if the password is too short, and disapperars if minimum 6 characters entered. 
If any of the input altered the error message reappeared if input invalid.

The submit button will be disabled until all fields are correctly validated. 

Technologies Used:
1. Html
2. Tailwind CSS
3. Java Script
